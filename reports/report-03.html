二、主要资讯梳理

1. 供应端：蒙古新增矿区投产预期压制远期，但山西现实供应仍偏紧

东方财富6月22日期市收评中提到，蒙古国南戈壁省塔旺陶勒盖矿群的博尔特格矿区正式启动开采。按照规划，该矿区建成投产后年产煤炭能力将达到 1500万吨；目前“额尔登斯·塔旺陶勒盖”股份公司年产煤炭约 3500万吨，博尔特格矿区投产后公司年产量预计增至 5000万吨。

这条消息对焦煤的影响偏中长期利空：

  • 蒙古煤炭供给潜在增加；
  • 未来口岸蒙煤供应弹性增强；
  • 市场会担心进口蒙煤对国内焦煤价格形成压制。

但短期现实供应端并未完全宽松。五矿期货认为，焦煤主要矛盾仍在于“5.22矿难”后续影响，山西仍面临严格安监和表外产量收缩。本周焦煤产量继续下滑，铁水处于高位，焦煤供求结构短期仍偏紧。

因此，6月22日供应端是典型的“现实偏紧、预期宽松”：

山西焦煤现实供应仍偏紧，支撑现货； 蒙古矿区投产、蒙煤供应预期和交割仓单压力压制盘面。

2. 蒙煤端：ER公司蒙3#精煤竞拍成交价小幅下跌

东方财富6月22日期市收评引用 Mysteel 数据称，6月22日蒙古国 ER 公司炼焦煤线上竞拍，蒙3#精煤 A11、V20-28、S0.85、G75、Mt10 起拍价 950元/吨，挂牌数量 1.28万吨 全部成交，成交价格 1010元/吨，较6月19日下跌 5元/吨。

该价格为不含税价格，供货地点为中国甘其毛都口岸监管区，供货时间为付款后90天内，最后供应日期为 2026年9月20日。

这条信息对盘面的含义是：

  1. 蒙煤竞拍全部成交，说明低价资源仍有需求；
  2. 成交价格较上期下跌，说明蒙煤端边际不强；
  3. 蒙煤作为期货交割和替代资源，会对焦煤盘面形成压制；
  4. 但跌幅仅5元/吨，更多是弱势信号，不是大幅崩塌。

3. 需求端：铁水高位仍支撑刚需，但淡季负反馈预期抬头

五矿期货认为，铁水继续处于高位，焦煤供求结构短期偏紧，这一点体现在山西主焦煤现货和海运煤价格持续上涨上。

但需求端的隐忧同样明显：

  • 焦炭多轮提涨侵蚀钢厂利润；
  • 建材处于高温、梅雨淡季；
  • 市场开始提前交易钢厂利润压缩后的减产预期；
  • 黑色板块整体风险偏好下降。

因此，6月22日需求端可以概括为：

现实需求未明显塌陷，铁水仍支撑刚需； 但远期需求预期转弱，盘面提前反映钢材淡季和利润压缩压力。

4. 焦炭端：第八轮提涨于6月22日落地，但期货回落反映负反馈预期

东方财富6月24日转载证券时报报道中披露，焦炭第八轮涨价已于 6月22日0时起落地执行。主流焦企自6月17日发起第八轮提涨，原计划20日执行，钢厂端推迟两日至22日执行。报道中还提到，现货市场两次提涨落地，累计上涨 100—110元/吨；自4月以来焦炭现货累计落地八轮提涨，累计上涨 400—440元/吨。

6月22日当天，虽然焦炭现货第八轮提涨落地，但焦炭期货 J2609 收跌2.58%。这说明盘面不再单纯交易焦炭现货提涨，而是开始交易：

  • 焦炭提涨侵蚀钢厂利润；
  • 钢厂对后续提涨接受能力下降；
  • 高铁水未来可能不可持续；
  • 期货资金从“供给短缺”逻辑切换至“淡季需求负反馈”逻辑。

对焦煤来说，这意味着焦炭端短期仍能提供成本传导支撑，但中期会带来钢厂利润压缩后的反向压力。

三、价格驱动逻辑

1. 盘面受蒙煤和仓单压力拖累

五矿期货指出，焦煤短期面临蒙煤以及交割仓单压力。近年来，蒙煤高发运和“科技煤”仓单一直是压在焦煤期货价格上方的重要因素。焦煤价格短期快速拉高后，盘面升水蒙煤，加上近月异常高持仓，交割压力向下拖拽盘面价格。

2. 山西主焦煤现货强，限制期货下跌空间

尽管盘面偏弱，但山西低硫主焦折盘面仍升水09合约 529.5元/吨，中硫主焦升水 417.5元/吨。这种大基差说明期货价格已经反映较多悲观预期，下方继续深跌需要现货端明显松动配合。

3. 5.22矿难后续影响仍在，供应端没有完全恢复

五矿期货认为，焦煤核心矛盾仍是5.22矿难后续影响，山西仍面临严厉安监和表外产量收缩。本周焦煤产量继续下滑，支撑主焦煤现货。

4. 蒙古矿区投产和竞拍价格小跌强化供应宽松预期

博尔特格矿区启动开采带来中长期供应增加预期；蒙3#精煤竞拍成交价较上期下跌5元/吨，说明蒙煤端边际偏弱。两者共同压制盘面情绪。

5. 焦炭现货提涨落地，但期货交易需求负反馈

焦炭第八轮提涨落地对焦煤成本传导偏利多；但焦炭期货下跌，说明市场担忧钢厂利润压缩和高铁水难以长期维持。这种预期会限制焦煤继续反弹的高度。

四、风险点

1. 蒙煤发运继续高位

如果蒙古煤发运保持高强度，口岸库存继续累积，蒙煤将继续压制焦煤盘面估值。

2. 交割仓单压力继续释放

焦煤期货近月合约此前已面临交割压力，如果仓单数量继续增加，远月主力也会受到估值拖累。

3. 钢材淡季负反馈加剧

高温、梅雨和建材需求疲弱可能继续压缩钢厂利润，进而削弱铁水和焦煤需求。

4. 山西煤矿复产加快

如果山西煤矿复产速度快于预期，前期供给收缩溢价将继续回吐。

5. 焦炭提涨难以继续传导

第八轮提涨虽已落地，但如果后续第九轮或更高轮次提涨受阻，焦企利润难以修复，焦煤采购积极性将受到影响。

五、综合判断

6月22日焦煤期货处于明显的矛盾博弈阶段：

现货端：山西主焦煤偏紧、基差大、供应现实偏强； 期货端：蒙煤、仓单、复产预期和钢材淡季负反馈共同压制。

短期判断：震荡整理，盘面仍承压

焦煤2609盘中触及阶段低位后修复，说明下方有一定承接。但蒙煤和仓单压力未解除，且焦炭期货明显回落，短期盘面仍难以快速转强。

中期判断：关注蒙煤发运和山西供应恢复节奏

如果蒙煤发运回落、口岸库存去化，同时山西供应仍受安监影响，焦煤中期仍可能重新走强。反之，如果蒙煤高发运延续、山西复产加快、钢厂利润继续压缩，盘面将维持偏弱震荡。

核心矛盾

当前焦煤核心矛盾是：

山西安监和5.22矿难后续影响导致主焦煤现货强势 对抗 蒙煤供应预期、交割仓单压力、焦炭期货回落和钢材淡季负反馈。

七、结论

6月22日焦煤期货的核心特征是：

现货强、盘面弱； 基差大、仓单压制强； 现实供应偏紧，但远期供应和需求预期转弱。

利多因素包括：

  1. 山西主焦煤现货升水仍大；
  2. 5.22矿难后续安监影响仍在；
  3. 焦煤产量阶段性下滑；
  4. 铁水仍处高位；
  5. 焦炭第八轮提涨落地。

利空因素包括：

  1. 蒙煤竞拍价格小幅下跌；
  2. 蒙古新矿区投产带来中长期供应增加预期；
  3. 交割仓单压力仍大；
  4. 焦炭期货明显回落；
  5. 钢材淡季和钢厂利润压缩引发负反馈预期。

综合判断：

6月22日焦煤处于“基差支撑下的偏弱震荡”阶段。短期盘面仍受蒙煤和仓单压制，中期能否重新转强，关键取决于蒙煤发运是否回落、山西供应是否继续偏紧，以及焦炭提涨能否顺利向钢厂传导。
