焦煤期货资讯分析报告

资讯日期：2026年7月2日 品种：大商所焦煤 / 焦煤主力 JM2609 / 焦煤主连 数据来源：东方财富焦煤期货页、曲合期货、SMM、华泰期货转引资讯、生意社、兰格钢铁网等公开信息 整理日期：2026年7月3日

说明：东方财富焦煤期货页为动态行情页，历史行情和资讯需要结合期货行情平台、期货公司观点和产业资讯交叉核验。7月2日盘面主口径采用曲合期货/市场盘后资讯：焦煤期货主力合约收涨0.20%，收报1267.5元/吨；当日最高1278.0元/吨，最低1252.5元/吨，持仓增加4855手至444101手。SMM焦煤页面显示焦煤主连价格1278.5元/吨、涨1.0672%，与盘后收盘口径存在差异，本文以1267.5元/吨作为收盘主口径。

一、当日核心行情

7月2日，焦煤主力 JM2609 小幅反弹。盘后资讯显示，焦煤期货主力合约 收涨0.20%，收报1267.5元/吨；当日最高价 1278.0元/吨，最低价 1252.5元/吨，持仓量增加 4855手 至 444101手。

SMM焦煤页面显示，焦煤主连价格为 1278.5元/吨，较前值上涨 13.5元/吨，涨幅 1.0672%；页面显示开盘 1275元/吨，昨收 1267.5元/吨，最高 1283.5元/吨，最低 1270元/吨。该口径更接近行情滚动或分时口径，与盘后收盘价略有差异，但方向均显示焦煤较7月1日出现修复。

焦炭方面，7月2日焦炭期货主力合约 收跌0.96%至1905.0元/吨，当日最高 1922.5元/吨，最低 1887.5元/吨，持仓减少 1408手 至 62296手；资金流向方面，焦炭期货资金整体流出 3652.48万元。

整体看，7月2日双焦盘面呈现分化：

焦煤小幅修复，焦炭继续回落；焦煤盘面有低位反弹，但焦炭走弱和钢材利润压力仍限制焦煤上方空间。

二、主要资讯梳理

1. 供应端：蒙古取消罢工，口岸扰动缓和，供应利多减弱

7月1日市场曾关注甘其毛都口岸通关扰动，部分蒙古国司机聚集导致境外车辆堵塞，通关预计降至800车以下。7月2日，华泰期货相关资讯标题为“蒙古取消罢工，双焦价格回调”，说明口岸扰动边际缓和。

该信息对焦煤偏利空：

  1. 前一日通关扰动利多有所消退；
  2. 蒙煤供给中断预期下降；
  3. 口岸市场成交仍偏冷清；
  4. 蒙5原煤价格弱稳在1150元/吨左右。

因此，7月2日供应端不是单纯偏紧，而是：

国内煤矿供给恢复仍慢，但蒙煤通关扰动边际缓和，削弱短线供应利多。

2. 国内煤矿端：523家煤矿产量继续下滑，但精煤库存回升

华泰期货转引钢联数据称，523家煤矿产量继续下滑，精煤库存环比回升。这一结构比较关键：

指标 方向 含义
523家煤矿产量 继续下滑 供应端仍有支撑
精煤库存 环比回升 下游采购放缓或高价煤成交压力增加
主产区煤价 略有松动 高价资源接受度下降
口岸成交 冷清 蒙煤端拖累盘面

这说明焦煤供需结构并非全面转宽，但边际压力正在累积：

产量低支撑价格下方，库存回升和成交冷清压制价格上方。

现货端的核心含义是：

  1. 主焦煤价格没有全面松动，现货对盘面仍有支撑；
  2. 局部肥煤上涨，说明部分配煤资源仍有结构性支撑；
  3. 但主产区煤价略有松动、口岸成交冷清，说明高价煤继续上冲阻力加大；
  4. 期货反弹幅度有限，说明市场对现货支撑的定价仍谨慎。

4. 基差端：焦煤现货大幅升水期货，盘面下方仍有支撑

生意社焦煤现期图显示，截至 2026年7月2日，焦煤现货价格为 1911.25元/吨，主力合约约 1267元/吨，主力基差为 643.75元/吨，基差率 33.68%。生意社同时显示，焦煤180日平均基差率为 21.80%，90日平均基差率为 22.95%。

基差端对焦煤盘面的影响是：

现货大幅升水会限制焦煤期货继续深跌，但如果现货价格开始松动，基差支撑也会快速弱化。

5. 焦炭端：第九轮提涨已落地，第十轮提涨计划出现，但期货继续承压

华泰期货转引资讯称，焦炭第九轮提涨全面落地，累计涨幅 450—495元/吨；焦企计划第十轮提涨，焦炭利润有所修复。但由于原料煤不足叠加利润不佳，焦炭生产积极性有限，下游需求相对平稳，短期焦炭价格继续跟随焦煤波动。

焦炭端对焦煤的影响较为复杂：

利多方面：

  • 第九轮提涨全面落地，说明焦煤成本向焦炭端传导仍在；
  • 第十轮提涨计划出现，焦企仍希望继续修复利润；
  • 原料煤不足说明部分煤种仍有结构性支撑。

利空方面：

  • 焦炭期货7月2日收跌0.96%，说明资金并不认可持续提涨；
  • 钢厂利润承压，对高价焦炭和焦煤接受度下降；
  • 焦炭利润虽修复，但生产积极性仍有限，限制对焦煤的补库强度。

焦炭端结论：

现货提涨仍支撑焦煤成本，但焦炭期货走弱显示钢焦博弈和负反馈压力仍在。

三、价格驱动逻辑

1. 前期利空释放后，焦煤低位小幅修复

7月1日焦煤收跌且资金流出，市场已部分消化口岸扰动解除、黑色系走弱和高价煤成交转弱等利空。7月2日焦煤主力低位小幅反弹，更多属于技术性修复和基差支撑。

2. 大基差支撑期货下方

生意社显示焦煤主力基差仍高达643.75元/吨，基差率33.68%，远高于90日和180日平均基差率。期货深贴水状态下，盘面继续下跌需要现货明显转弱配合。

3. 蒙古取消罢工削弱短线供应题材

7月1日口岸堵车曾带来短期利多，但7月2日“蒙古取消罢工”使通关扰动预期减弱，蒙煤供应担忧回落，压制焦煤反弹高度。

4. 产量下滑与库存回升并存，供需矛盾未单边化

523家煤矿产量继续下滑，说明供应端仍有支撑；但精煤库存环比回升、口岸成交冷清，说明下游采购情绪谨慎。供需信号分化使盘面难以出现单边趋势。

5. 焦炭走弱限制焦煤上涨

焦炭期货主力7月2日收跌0.96%，资金流出3652.48万元。焦炭作为焦煤下游产品走弱，会削弱焦煤反弹持续性，尤其是在钢厂利润承压的背景下。

四、风险点

1. 蒙煤通关恢复

如果甘其毛都口岸通关恢复至高位，蒙煤供应扰动利多将消退，焦煤盘面可能重新承压。

2. 精煤库存继续回升

523家煤矿产量虽下滑，但精煤库存环比回升。如果后续精煤持续累库，说明下游采购不足，现货支撑会减弱。

3. 口岸成交持续冷清

蒙5原煤弱稳在1150元/吨左右，口岸成交冷清。如果蒙煤价格继续走弱，会压制焦煤期货估值。

4. 焦炭期货继续下跌

焦炭第九轮提涨落地，但期货仍下跌。如果焦炭继续走弱，说明市场更担心钢厂利润负反馈，会拖累焦煤。

5. 现货价格松动导致基差收敛

目前焦煤期货重要支撑来自大基差。如果现货价格下跌，而期货不涨，基差支撑将明显弱化。

五、综合判断

7月2日焦煤市场的核心特征是：

低位小幅修复，但反弹动能不足；现货和基差支撑仍强，但蒙煤扰动解除、库存回升和焦炭走弱压制上方。

短期判断：低位震荡

焦煤主力收涨0.20%，并非强势反转，而是低位修复。盘面仍处于供应偏紧和需求负反馈之间的拉锯阶段。1260元/吨附近有基差支撑，但1280元/吨上方仍面临压力。

中期判断：方向取决于现货是否松动

如果主焦煤现货继续坚挺，且煤矿产量恢复缓慢，焦煤期货仍有修复空间；如果精煤库存继续回升、蒙煤成交持续冷清、焦炭和钢材端继续转弱，则焦煤仍可能维持震荡偏弱。

核心矛盾

当前焦煤核心矛盾是：

主焦煤现货持稳、大基差、煤矿产量下滑 对抗 蒙古取消罢工、口岸成交冷清、精煤库存回升、焦炭期货走弱和钢厂利润压力。

七、结论

7月2日焦煤主力合约收涨0.20%，收报1267.5元/吨，属于前一日下跌后的低位修复。上涨的支撑来自：

  1. 现货主焦煤多数持稳；
  2. 生意社显示焦煤主力基差仍高达643.75元/吨；
  3. 523家煤矿产量继续下滑；
  4. 焦炭第九轮提涨已全面落地；
  5. 部分肥煤现货仍有上调。

但反弹并不强，原因在于：

  1. 蒙古取消罢工，口岸扰动利多减弱；
  2. 口岸市场成交仍冷清；
  3. 蒙5原煤弱稳在1150元/吨左右；
  4. 精煤库存环比回升；
  5. 焦炭期货收跌0.96%，资金流出；
  6. 钢厂利润压力仍制约高价煤和高价焦炭的接受度。

综合判断：

7月2日焦煤处于“高基差支撑下的低位震荡”阶段。短期下方有现货和基差支撑，但上方受蒙煤扰动解除、精煤库存回升和焦炭走弱压制。后续重点观察蒙煤通关、精煤库存、主焦煤现货是否松动，以及焦炭第十轮提涨能否真正落地。
