焦煤期货资讯分析报告

资讯日期：2026年7月1日 品种：大商所焦煤 / 焦煤主力 JM2609 / 焦煤主连 数据来源：东方财富焦煤期货页、东方财富期市直播、东方财富午盘播报、SMM、兰格钢铁网/新浪财经等公开信息 整理日期：2026年7月2日

说明：东方财富焦煤期货页为动态行情页，历史行情和资讯需结合东方财富期市直播、东方财富盘后资讯、SMM行情页及产业资讯交叉核验。7月1日主口径采用东方财富期市直播/南财盘后数据：焦煤主力合约收跌1.06%，收盘1262.0元/吨，日内资金流出6.7亿元。SMM焦煤页面显示焦煤主连价格1265元/吨、开盘1262元/吨、最高1271元/吨、最低1256.5元/吨，与盘后收盘口径略有差异，本文以东方财富盘后收盘价1262.0元/吨作为主口径。

一、当日核心行情

7月1日，焦煤主力合约明显回落。东方财富期市直播/南财智讯盘后数据显示，大连商品交易所焦煤主力合约 收跌1.06%，收盘报1262.0元/吨，日内资金流出6.7亿元。

午盘阶段，东方财富午盘播报显示，国内商品期货早盘收盘主力合约多数下跌，跌幅方面，焦炭、甲醇、焦煤跌超2%，黑色系整体偏弱。

SMM行情页面显示，焦煤主连当日价格为 1265元/吨，开盘 1262元/吨，昨收 1262元/吨，最高 1271元/吨，最低 1256.5元/吨。与东方财富盘后1262.0元/吨相比，SMM页面更接近行情滚动或分时口径，二者方向一致，均显示7月1日焦煤盘面弱于6月30日。

整体看，7月1日焦煤盘面表现为：

前一日反弹后转弱，日内资金继续流出；口岸扰动带来短线供应题材，但黑色系整体走弱、资金流出和高价煤成交转弱压制盘面。

二、主要资讯梳理

1. 供应端：甘其毛都口岸通关扰动带来短线支撑，但市场没有转为强势

东方财富午盘播报引用 Mysteel 消息称，7月1日07时，部分蒙古国司机再次聚集在查干哈达堆煤场至甘其毛都口岸运输公路上，造成甘其毛都口岸境外车辆堵塞。Mysteel与口岸贸易企业沟通后获悉，甘其毛都口岸通关尚未完全停止，但进车缓慢，预计当日通关降至 800车以下。

这条消息对焦煤理论上偏利多，因为甘其毛都口岸是蒙煤进入中国的重要通道。通关下降会影响蒙煤短期到货节奏，尤其会影响蒙5#、蒙3#等资源的供应预期。

但从盘面反应看，焦煤并未因此转强，反而收跌，说明市场更关注以下压力：

  1. 前期反弹后资金兑现；
  2. 进口蒙煤供应只是短期扰动，并未完全停通；
  3. 下游对高价煤接受度下降；
  4. 黑色系整体偏弱，焦炭同步下跌；
  5. 钢材淡季和钢厂利润压力仍在。

因此，供应端可以概括为：

口岸扰动构成短线利多，但由于通关未完全停止，且盘面资金流出，利多未能推动焦煤持续上涨。

2. 需求端：铁水仍处高位，但边际压力开始显现

7月1日需求端的主线仍然是“现实刚需尚可、预期压力增加”。

焦炭市场资讯显示，钢厂高炉开工仍处高位，日均铁水产量处于高位，钢厂对焦炭刚需尚可。这意味着焦煤作为焦炭原料，需求端仍有底部支撑。

但需求端的问题同样突出：

  • 钢材淡季压力仍在；
  • 南方梅雨季影响终端施工；
  • 钢厂利润被焦炭多轮提涨压缩；
  • 焦炭现货虽然偏强，但期货下跌说明市场担忧后续钢厂承接能力；
  • 黑色系午盘集体走弱，焦炭、焦煤均跌超2%。

所以需求端不是“需求塌陷”，而是：

铁水高位支撑现实需求，但钢厂利润和成材需求压制远期预期。

  1. 主焦煤和肥煤多数持稳，说明优质煤种现货并未全面松动；
  2. 临汾瘦煤下跌50元/吨，说明部分配煤资源和非核心高价煤开始承压；
  3. 期货下跌幅度大于主焦煤现货，基差支撑仍可能存在；
  4. 但高价资源成交转弱，会限制现货继续上行空间。

4. 焦炭端：第九轮提涨全面落地，但期货市场交易负反馈

7月1日国内焦炭主流市场暂稳运行。兰格钢铁网/新浪财经资讯显示，焦炭第九轮提涨全面落地，焦企利润有所修复，焦企生产较为稳定，下游钢厂需求尚可，焦企出货顺畅，焦企多保持低库存运行。

焦炭第九轮提涨全面落地，对焦煤有两方面影响：

利多：

  • 焦企利润修复；
  • 焦企生产稳定；
  • 钢厂刚需仍在；
  • 焦企低库存运行，成本支撑仍较强。

利空/压力：

  • 焦炭期货和焦煤期货均下跌，说明资金交易的是后续钢厂利润负反馈；
  • 若第十轮提涨无法顺利落地，焦企利润修复空间有限；
  • 钢厂利润继续被压缩后，可能通过压价原料或控制采购来反馈到焦煤端。

因此，焦炭端结论是：

现货提涨利多焦煤成本支撑，但期货市场已经开始交易提涨后的利润压力和需求负反馈。

5. 资金与盘面：下跌伴随资金流出，反弹持续性被削弱

7月1日焦煤收跌1.06%，日内资金流出6.7亿元。结合6月30日焦煤上涨但资金流出3.79亿元的情况看，焦煤连续两个交易日资金偏流出。

这说明：

  1. 6月29日大涨后的追涨资金不足；
  2. 6月30日反弹已经有兑现迹象；
  3. 7月1日资金继续撤出，盘面主动上攻动力减弱；
  4. 供应题材仍在，但资金更重视黑色系整体负反馈风险。

从盘面性质看，7月1日更像是：

供应扰动背景下的冲高后回落，而不是基本面突然转空。

三、价格驱动逻辑

1. 黑色系整体走弱带动焦煤回落

午盘阶段，焦炭、焦煤均跌超2%，铁矿石也偏弱，黑色系整体承压。焦煤不是孤立下跌，而是跟随黑色产业链风险偏好下降。

2. 资金流出削弱反弹持续性

焦煤主力当日资金流出6.7亿元，说明市场对6月29日以来的反弹并不愿持续追高。资金面偏弱是当日价格下行的重要原因。

3. 口岸扰动带来利多，但未能改变盘面方向

甘其毛都口岸通关预计降至800车以下，对蒙煤短期供应偏利多。但通关未完全停止，且扰动可能属于短期事件，因此市场没有给予持续上涨定价。

4. 焦炭第九轮提涨落地支撑成本，但负反馈预期升温

焦炭现货提涨全面落地，对焦煤成本端有支撑；但钢厂利润继续承压，市场担心后续提涨难以继续传导，因此期货端反而偏弱。

5. 主焦煤现货稳定，限制期货深跌空间

SMM显示多数主焦煤、肥煤价格持稳，说明现货端没有全面松动。期货下跌后，若现货继续坚挺，基差支撑会限制焦煤继续下跌。

四、风险点

1. 蒙煤通关扰动快速解除

如果甘其毛都口岸堵车问题很快缓解，通关恢复至高位，短线供应扰动利多会消退，焦煤盘面可能继续承压。

2. 资金继续流出

7月1日焦煤资金流出6.7亿元。如果后续继续流出，说明多头信心不足，盘面反弹难以延续。

3. 高价煤成交继续转弱

虽然主焦煤现货多数持稳，但若后续竞拍继续流拍或跌价，现货支撑会减弱。

4. 钢厂利润继续压缩

焦炭第九轮提涨全面落地后，钢厂成本上升。如果钢厂利润进一步恶化，铁水可能回落，焦煤需求端将转弱。

5. 焦炭第十轮提涨不确定性

市场有第十轮提涨预期，但落地难度较第九轮更大。若第十轮受阻，焦企利润修复预期会降温，焦煤采购情绪可能走弱。

五、综合判断

7月1日焦煤市场的核心特征是：

现货仍有支撑，口岸有短线扰动，但盘面资金流出，黑色系整体转弱。

短期判断：震荡偏弱，关注1260附近支撑

焦煤主力收盘1262.0元/吨，接近SMM显示的日内低位区间。短期若蒙煤通关扰动持续、主焦煤现货保持坚挺，1260附近可能有支撑；但资金继续流出和黑色系转弱会压制反弹高度。

中期判断：供应题材未消失，但上方空间取决于钢厂利润和铁水

山西供应扰动、优质主焦煤结构性偏紧、焦炭提涨落地仍是中期支撑。但如果钢厂利润继续压缩、铁水开始下降，焦煤会从“供应驱动”转向“需求负反馈”定价。

核心矛盾

当前焦煤核心矛盾是：

甘其毛都通关扰动、主焦煤现货稳定、焦炭第九轮提涨落地 对抗 资金流出、黑色系走弱、钢厂利润承压和高价煤成交转弱。

七、结论

7月1日焦煤期货下跌，主要不是因为供应端利多消失，而是盘面在交易：

  1. 6月29日大涨后的资金兑现；
  2. 6月30日上涨中资金已流出，7月1日资金继续撤出；
  3. 黑色系整体转弱，焦炭、焦煤早盘跌超2%；
  4. 焦炭第九轮提涨落地后，钢厂利润压力加大；
  5. 高价煤成交转弱和需求负反馈预期增强。

但焦煤仍有支撑：

  1. 甘其毛都口岸通关预计降至800车以下；
  2. 主焦煤、肥煤多数现货价格持稳；
  3. 焦炭第九轮提涨全面落地；
  4. 焦企低库存运行，出货顺畅；
  5. 高炉开工和铁水仍处高位。

综合判断：

7月1日焦煤处于“供应题材仍在、资金和需求预期转弱”的震荡偏弱阶段。短期重点看1260元/吨附近支撑和甘其毛都通关恢复情况；中期重点看山西复产放量、钢厂利润、铁水产量和焦炭第十轮提涨能否继续传导。
