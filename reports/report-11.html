焦煤期货资讯分析报告

资讯日期：2026年6月26日 品种：大商所焦煤 / 焦煤主力 JM2609 / 焦煤主连 数据来源：东方财富焦煤期货页、东方财富相关资讯、五矿期货黑色建材早报等公开信息 整理日期：2026年6月30日

说明：东方财富焦煤期货页为动态行情页，公开网页端可直接读取的信息有限；本报告以东方财富 6月26日相关资讯为主，盘面收盘价与基差信息使用五矿期货公开早报进行补充核验。

一、当日核心行情

6月26日，焦煤主力合约 JM2609 日盘小幅反弹。五矿期货黑色建材早报显示，6月26日焦煤主力 JM2609 合约日内收涨0.81%，收盘报1248.5元/吨。

现货端仍明显强于盘面：

现货口径 现货价格 折盘面仓单价 相对09合约升水
山西低硫主焦 1994.8元/吨 1759元/吨 565.5元/吨
山西中硫主焦 1700元/吨 1693元/吨 444.5元/吨
乌不浪金泉工业园区蒙5#精煤 1358元/吨 1333元/吨 84.5元/吨

焦炭方面，6月26日焦炭主力 J2609 合约日内收涨0.72%，收盘报1946元/吨。夜盘阶段，东方财富转载的每日经济新闻消息显示，6月26日夜盘收盘时，焦煤、焦炭涨超1%，说明双焦盘面在当日晚间延续修复。

总体看，6月26日焦煤盘面属于：

日盘小幅修复 + 现货升水仍大 + 夜盘双焦继续反弹

这不是单纯的现货下跌驱动，而更像是前期连续回落后的估值修复。

二、主要资讯梳理

1. 供应端：矿山产能利用率下降，供应端仍有支撑

东方财富期货早餐引用 Mysteel 数据显示，截至6月25日当周，523家炼焦煤矿山样本核定产能利用率为68.2%，环比下降3.0个百分点；原煤日均产量 153.1万吨，环比减少6.8万吨；精煤日均产量 67.1万吨，环比减少1.4万吨。

库存方面，原煤库存 443.3万吨，环比减少7.8万吨；精煤库存 173.9万吨，环比增加3.0万吨。

这组数据说明两点：

  1. 矿山端产量继续下滑，供应端对焦煤仍有支撑；
  2. 精煤库存小幅累积，说明部分煤种销售或下游接货节奏有所放缓。

同时，东方财富午盘播报提到，6月26日新疆地区煤矿正常生产销售，产量整体稳定，发运正常；近期陕蒙煤价小幅上调，且因新疆铁路调整幅度较小，疆煤发往部分省份仍有优势。

因此，供应端不是单向收紧，而是呈现结构分化：

山西等核心焦煤产区受安监和复产节奏影响偏紧； 新疆等区域生产稳定，对部分下游形成补充供应。

2. 需求端：铁水仍高，但钢厂利润和成材需求走弱

东方财富午盘播报引用 Mysteel 数据显示，247家钢厂高炉开工率 84.41%，环比增加 0.16个百分点；高炉炼铁产能利用率 91.08%，环比增加 0.27个百分点；日均铁水产量 242.95万吨，环比增加 0.71万吨。

这说明焦煤、焦炭的刚需仍有支撑。

但同一数据也显示，钢厂盈利率为 51.08%，环比下降 4.76个百分点。也就是说，钢厂仍在生产，但利润已经被压缩。

成材端更弱。东方财富期货早餐引用 Mysteel 数据显示，截至6月25日当周：

指标 数值 环比变化
螺纹产量 213.25万吨 -5.48万吨，-2.51%
螺纹表需 188.75万吨 -32.05万吨，-14.52%
螺纹厂库 196.25万吨 +11.33万吨，+6.13%
螺纹社库 487.14万吨 +13.17万吨，+2.78%

这说明下游钢材需求进入淡季压力区间，螺纹表需大幅回落、厂库和社库同步增加。对焦煤来说，这是较明显的需求端利空。

因此，需求端可以概括为：

铁水高位支撑焦煤刚需，但钢厂利润下降和螺纹需求走弱限制上行空间。

3. 库存端：港口焦煤累库，焦炭港口去库

东方财富期市收评引用 Mysteel 数据显示，6月26日当周，全国16个港口进口焦煤库存为 567.70万吨，环比增加13.30万吨。

焦煤港口库存上升，说明进口煤端供应压力有所增加，尤其是华北和华南港口累库，对盘面上方构成压制。

焦炭方面，全国18个港口焦炭库存为 288.20万吨，环比减少5.05万吨。

分区域看：

焦炭港口去库对焦炭价格有一定支撑，也会通过焦化利润和焦企补库意愿间接影响焦煤需求。

库存端的结论是：

焦煤端：港口累库，偏利空； 焦炭端：港口去库，偏利多； 双焦库存信号分化，焦煤上行需要更多供应收缩或需求改善配合。

4. 焦炭端：第九轮提涨进入执行和博弈阶段

证券时报在东方财富发布的报道中提到，主流焦企已发起第九轮提涨，湿熄焦上调 50元/吨、干熄焦上调 55元/吨，自 6月26日0时起执行。

五矿期货早报则判断，焦炭第九轮提涨“阶段性落地，部分仍在博弈”。这说明焦炭现货成本支撑仍在，但钢厂利润下降使后续提涨继续传导的难度加大。

焦炭端对焦煤的影响是双向的：

  • 第九轮提涨若顺利落地，焦企利润修复，有利于焦煤采购；
  • 如果钢厂利润继续被压缩，钢厂对焦炭提涨抵触增强，焦企对高价煤采购会变得谨慎。

三、价格驱动逻辑

1. 供应端仍偏紧，支撑盘面修复

矿山样本产能利用率下降、原煤和精煤日均产量下滑，说明供应端并未快速恢复。山西等核心焦煤产区仍受安监和复产节奏影响，主焦煤现货继续强于期货盘面。

2. 现货升水较大，期货估值偏低

6月26日山西低硫主焦折盘面仓单价较JM2609升水 565.5元/吨，山西中硫主焦升水 444.5元/吨。大基差说明盘面已经较大程度反映了需求弱和复产预期，短期容易出现修复。

3. 铁水高位维持刚需

247家钢厂日均铁水产量达到 242.95万吨，仍处高位，对焦炭和焦煤形成刚需支撑。只要铁水未明显下降，焦煤价格下方仍有支撑。

4. 港口焦煤累库和螺纹表需走弱压制上方空间

焦煤港口库存增加、螺纹表需大幅下降、钢厂盈利率下滑，说明焦煤反弹并不是需求全面走强，而更偏向估值修复和供应扰动支撑。

5. 夜盘双焦继续反弹，短线情绪有所修复

东方财富消息显示，6月26日夜盘焦煤、焦炭涨超1%。这说明日盘反弹后，市场短线情绪继续修复，但是否形成持续上涨，还需要看下周供应、铁水和焦炭提涨落地情况。

四、风险点

1. 钢材需求继续走弱

螺纹表需环比下降 14.52%，同时厂库、社库同步上升。如果钢材淡季压力继续扩大，钢厂利润进一步下降，铁水高位可能难以维持。

2. 港口进口焦煤库存继续累积

全国16港进口焦煤库存环比增加 13.30万吨。如果后续蒙煤通关和海运煤到港维持高位，焦煤港口库存继续累积，会削弱主焦煤现货强势对盘面的支撑。

3. 焦炭提涨传导受阻

第九轮提涨已经进入执行阶段，但部分仍在博弈。如果钢厂抵触加重，焦企利润修复受限，会影响焦企补库积极性。

4. 煤矿复产节奏超预期

如果停产煤矿复产进度明显快于市场预期，供应端利多会减弱，盘面可能再次承压。

5. 宏观和黑色板块情绪偏弱

五矿期货提到，黑色板块近期受到宏观数据偏弱、海外加息预期和传统消费淡季等因素压制。若板块整体风险偏好继续下降，焦煤即使基本面偏紧，也可能被板块情绪拖累。

五、综合判断

6月26日焦煤期货的主要特征是：

供应端偏紧仍在，现货升水较大，盘面估值偏低； 但钢材需求走弱、港口焦煤累库、钢厂利润下降限制反弹高度。

短期判断：震荡修复

焦煤主力在前期连续回落后，6月26日出现小幅反弹，夜盘双焦继续走强。考虑到现货升水较大、铁水仍处高位，短期焦煤继续下跌的压力有所缓和。

中期判断：上方仍受需求压制

中期能否继续走强，关键看焦煤供应是否继续收缩，以及铁水是否能维持高位。如果螺纹需求继续走弱、钢厂利润继续下降，焦煤反弹空间会被压制。

核心矛盾

当前焦煤市场的核心矛盾是：

供应端安监扰动和主焦煤偏紧 对抗 成材淡季、钢厂利润下降和进口焦煤港口累库

七、结论

6月26日焦煤期货的上涨更偏向于“估值修复 + 供应扰动支撑”，而不是需求端全面转强。

利多因素包括：

  1. 矿山产能利用率下降；
  2. 原煤和精煤产量下降；
  3. 铁水仍在240万吨以上高位；
  4. 山西主焦煤现货大幅升水盘面；
  5. 焦炭第九轮提涨进入执行阶段。

利空因素包括：

  1. 螺纹表需明显下滑；
  2. 螺纹厂库、社库同步增加；
  3. 钢厂盈利率下降；
  4. 港口进口焦煤库存增加；
  5. 焦炭提涨仍存在钢焦博弈。

综合来看：

短期焦煤偏震荡修复； 中期上方空间取决于铁水能否维持高位，以及山西煤矿复产是否继续慢于预期。

对于后续跟踪，最重要的是看三组变量：

  1. 供应端：矿山产能利用率、停产/复产、蒙煤通关；
  2. 需求端：铁水、钢厂利润、螺纹表需；
  3. 库存端：港口焦煤库存、焦炭港口库存、焦企原料库存。
