焦煤期货资讯分析报告

资讯日期：2026年6月24日 品种：大商所焦煤 / 焦煤主力 JM2609 / 焦煤主连 数据来源：东方财富焦煤期货页、东方财富相关资讯、五矿期货、光大期货、Mysteel转引资讯等公开信息 整理日期：2026年6月30日

说明：东方财富焦煤期货页为动态行情页，历史行情和部分资讯需要通过东方财富期货频道、东方财富转载机构观点及公开期货公司观点交叉核验。本报告以 2026年6月24日当日相关资讯及次日公开可验证的盘面数据为主。

一、当日核心行情

6月24日，焦煤主力 JM2609 继续偏弱运行。东方财富转载五矿期货观点显示，6月24日焦煤主力 JM2609合约日内收跌0.20%，收盘报1245.5元/吨。

现货端仍明显强于盘面：

现货口径 现货价格 折盘面仓单价 相对09合约升水
山西低硫主焦 1992.6元/吨 1812元/吨 566.5元/吨
山西中硫主焦 1700元/吨 1693元/吨 447.5元/吨
乌不浪金泉工业园区蒙5#精煤 1358元/吨 1333元/吨 87.5元/吨

焦炭方面，同日焦炭主力 J2609合约日内收涨1.17%，收盘报1947.5元/吨。日照港准一级湿熄焦报价 1760元/吨，折盘面仓单价 2037.5元/吨，升水09合约 90元/吨；吕梁准一级干熄焦报价 1955元/吨，折盘面仓单价 2174元/吨，升水09合约 226.5元/吨。

从盘面表现看，6月24日焦煤、焦炭出现分化：

焦煤期货继续小幅回落，焦炭期货反弹； 但焦煤现货升水仍大，说明盘面下跌更多反映蒙煤、交割仓单和需求预期压力，而不是山西主焦煤现货全面转弱。

二、主要资讯梳理

1. 供应端：山西焦煤供给持续偏紧，煤矿“复产不复量”

6月24日晚间东方财富转载财联社/Mysteel消息称，焦煤供给持续偏紧，前期山西煤矿安全整治导致大量炼焦煤产能停产，目前复产呈现“复产不复量”特征，复工矿井多为低负荷生产。优质低硫主焦、肥煤货源紧缺，产地煤价持续攀升，焦化入炉成本中枢继续上移。

关键数据如下：

指标 数值
5月23日起山西炼焦煤停产煤矿 155座
涉及产能 18310万吨
截至6月17日已复产煤矿 97座
已复产产能 11740万吨
仍停产煤矿 58座
仍停产产能 6570万吨

这说明供应端现实仍偏紧，尤其是低硫主焦、肥煤等关键配煤资源尚未完全恢复。对焦煤价格来说，这是核心支撑。

但需要注意，市场同时在交易“复产继续推进”的预期。也就是说：

现实：供应偏紧； 预期：复产推进、边际供应可能恢复。

这种现实与预期的矛盾，是6月24日焦煤期货没有跟随现货强势继续上涨的重要原因。

2. 需求端：铁水仍处高位，但钢材淡季压力加大

光大期货6月24日矿钢煤焦日报指出，焦煤需求端仍有一定支撑，焦企利润转好提升生产积极性，铁水产量维持高位运行，对原料需求有支撑；但终端需求偏弱，市场预期原料边际宽松。

五矿期货次日复盘也提到，铁水继续处于高位，焦煤供求结构短期仍维持偏紧；但黑色系整体面对淡季、高温多雨影响施工、钢材外需压力等因素，需求端有收缩压力。

因此，需求端并不是单边利多，而是：

方向 表现 对焦煤影响
刚需支撑 铁水维持高位、焦企利润改善 利多
终端压力 建筑钢材淡季、高温多雨影响施工 利空
钢厂利润 钢厂盈利仍有但收缩 中性偏空
预期变化 市场担心原料边际宽松 利空

综合看，6月24日焦煤需求端处于“短期刚需仍在，但中期负反馈风险上升”的状态。

3. 库存端：公开资讯未给出完整库存表，但结构信号偏分化

6月24日东方财富相关资讯没有披露完整的钢厂、焦企、港口和矿山库存明细。但结合当日资讯可以判断库存结构有三点：

  1. 山西主焦煤现货升水大，说明优质资源仍偏紧；
  2. 蒙煤和交割仓单对盘面形成压力，说明部分可交割或替代资源并不紧缺；
  3. 焦炭端部分钢厂焦炭库存下降，采购需求较高。

后续需要继续跟踪：

  • 523家炼焦煤矿山原煤、精煤库存；
  • 16港进口焦煤库存；
  • 焦化厂炼焦煤库存；
  • 247家钢厂炼焦煤库存；
  • 焦炭港口库存；
  • 钢厂焦炭库存。

库存端结论是：

优质主焦煤偏紧支撑现货； 蒙煤和仓单压力压制期货； 焦炭库存下降对焦炭和焦煤需求有一定支撑。

4. 焦炭端：第九轮提涨发布，成本支撑继续向下游传导

6月24日，市场消息显示焦炭价格偏强运行。主流焦企发起第九轮提涨，湿熄焦上调 50元/吨、干熄焦上调 55元/吨，自 6月26日0时起执行。

焦炭提涨的背景是：

  • 低硫优质主焦煤仍然紧缺；
  • 焦企生产成本居高不下；
  • 部分焦企因高价煤压力逐步减产，以减少优质主焦、肥煤等高价煤采购；
  • 焦炭供应有所收紧；
  • 部分钢厂焦炭库存下降，采购需求较高。

焦炭端对焦煤的影响有两层：

  1. 利多焦煤： 焦炭提涨说明焦煤成本压力正在向下游传导，焦企仍有补库和生产需求；
  2. 潜在利空： 如果焦炭继续提涨导致钢厂利润进一步压缩，后续可能触发钢厂抵触甚至负反馈。

因此，第九轮提涨本身短期偏利多，但也提高了后续钢焦博弈风险。

三、价格驱动逻辑

1. 供应端偏紧支撑现货

山西焦煤停产煤矿仍有58座、涉及产能6570万吨，且已复产矿井多低负荷生产，说明供应恢复并不充分。优质低硫主焦、肥煤紧缺，使山西主焦煤现货继续大幅升水盘面。

2. 蒙煤和交割仓单压力压制盘面

五矿期货指出，焦煤短期面临蒙煤以及交割仓单压力。近年来蒙煤高发运和相关仓单一直压制焦煤期货上方空间。6月24日虽然山西现货偏强，但蒙煤和仓单压力使盘面难以单边跟涨。

3. 高铁水仍维持焦煤刚需

铁水继续处于高位，焦企利润转好，支撑焦煤刚需。只要铁水不明显下降，焦煤价格下方仍有一定基本面支撑。

4. 终端钢材淡季和宏观压力限制上方空间

五矿期货提到，南方暴雨、高温拖累线下施工，建筑用钢需求预期持续走弱，同时海外主要央行加息、美联储鹰派信号、美债与美元走强压制工业品估值。宏观和成材需求压力使黑色系整体偏弱。

5. 焦炭第九轮提涨强化成本逻辑

焦炭第九轮提涨通知发布，说明焦煤成本压力仍在向焦炭端传导。焦炭主力当日上涨1.17%，也反映焦炭端短期相对强于焦煤。

四、风险点

1. 煤矿复产继续推进

虽然当前复产不复量，但如果后续山西煤矿低负荷生产逐渐恢复至正常水平，供应端利多会弱化。

2. 蒙煤高发运和仓单压力持续

如果蒙煤发运维持高位，口岸库存不能有效去化，期货盘面上方压力仍难消除。

3. 终端钢材需求继续走弱

高温、多雨、淡季施工放缓，以及钢材出口压力上升，可能削弱钢厂利润和铁水持续性，进而压制焦煤需求。

4. 焦炭提涨无法顺利落地

第九轮提涨计划自6月26日执行，但如果钢厂利润收缩加剧，提涨落地不顺，会影响焦企采购焦煤意愿。

5. 宏观风险偏好下降

海外加息预期、美元走强、商品指数破位下跌等因素，可能导致焦煤基本面偏强但盘面仍被资金情绪压制。

五、综合判断

6月24日焦煤市场的核心特征是：

现货强、期货弱； 供应现实偏紧、盘面交易复产和蒙煤压力； 焦炭偏强、焦煤受仓单和需求预期压制。

短期判断：震荡偏弱，但下方有现货升水支撑

焦煤主力JM2609收跌0.20%，盘面延续回落。但山西低硫主焦折盘面仍升水09合约566.5元/吨，中硫主焦升水447.5元/吨，说明盘面估值已经偏低，下方不宜过度看空。

中期判断：关注价格向上拐点，但需要等待压力释放

五矿期货认为当前盘面估值已经偏低，后续若蒙煤发运回落、口岸库存去化、山西供应影响持续，焦煤仍可能重新转强。但在此之前，盘面仍需要消化蒙煤、仓单和成材淡季压力。

核心矛盾

当前焦煤的核心矛盾是：

山西安监后供应偏紧、低硫主焦和肥煤紧缺 对抗 蒙煤/仓单压力、终端钢材淡季、复产预期和宏观工业品估值下移。

七、结论

6月24日焦煤期货下跌，并不是因为焦煤现货基本面全面转弱，而是盘面在交易：

  1. 蒙煤和交割仓单压力；
  2. 煤矿复产预期；
  3. 钢材淡季和宏观工业品估值下移；
  4. 焦煤前期上涨后的技术性回调。

但焦煤下方仍有支撑：

  1. 山西仍有58座炼焦煤矿未复产；
  2. 仍停产产能达6570万吨；
  3. 复产矿井多低负荷生产；
  4. 低硫主焦、肥煤货源紧缺；
  5. 铁水仍处高位；
  6. 焦炭第九轮提涨发布，成本传导仍在。

综合来看：

6月24日焦煤处于“现实偏强、预期承压”的震荡整理阶段。短期盘面仍受蒙煤、仓单和成材淡季压制，但现货升水和山西供应偏紧限制下跌空间。后续如果蒙煤发运回落、口岸库存去化、焦炭提涨顺利落地，焦煤存在重新转强的可能。
