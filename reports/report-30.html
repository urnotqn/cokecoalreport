焦煤期货资讯分析报告

资讯日期：2026年7月9日 品种：大商所焦煤 / 焦煤主力 JM2609 / 焦煤主连 数据来源：兰格钢铁网、东方财富午盘播报、SMM焦煤价格与期货行情、光大期货矿钢煤焦日报、建信期货焦炭焦煤日评、证券之星等公开信息 整理日期：2026年7月10日

说明：本报告重新按“焦煤期货资讯分析 Skill”独立整理，前序同日或前日版本不再作为基础。7月9日主口径采用兰格钢铁网盘后收盘数据：焦煤主力2609合约收盘 1290元/吨，较昨日结算价上涨 1元/吨，涨幅 0.08%；全天最高 1297.5元/吨，最低 1282元/吨，成交量 49.27万手。SMM页面显示的焦煤主连 1254元/吨、跌2.79% 更接近7月9日夜盘/滚动行情口径，不作为日盘收盘主口径。

一、当日核心行情

7月9日，焦煤主力 JM2609 日盘窄幅震荡，整体延续前一日反弹后的高位整理。兰格钢铁网盘后数据显示，焦煤期货主力2609合约全天最高 1297.5元/吨，最低 1282元/吨，最终收于 1290元/吨，较昨日结算价上涨 1元/吨，涨幅 0.08%，成交量 49.27万手。

证券之星同日资讯也显示，7月9日焦煤主力期货合约收盘上涨 0.08%，报 1290元/吨，成交金额 381.47亿元。这与兰格钢铁网收盘口径一致。

焦炭方面，证券之星显示7月9日焦炭主力期货合约收盘下跌 0.08%，报 1945元/吨，成交金额 65.57亿元。焦煤微涨、焦炭微跌，说明双焦并未形成明显单边共振，盘面更多处于震荡博弈阶段。

整体看，7月9日盘面可以概括为：

焦煤日盘小幅收涨，但涨幅只有0.08%，属于高位震荡整理；盘面没有继续强攻1300元/吨，上方压力仍在。

二、主要资讯梳理

1. 供应端：523家矿山产能利用率继续下降，供应端仍有支撑

东方财富午盘播报引用Mysteel数据称，本周 523家炼焦煤矿山样本核定产能利用率为66.1%，环比减少 0.9个百分点；原煤日均产量 148.5万吨，环比减少 1.9万吨；原煤库存 447.4万吨，环比增加 6.2万吨；精煤日均产量 62.5万吨，环比减少 2.7万吨；精煤库存 193.1万吨，环比增加 9.0万吨。

这组数据非常关键：

指标 数值 周环比 影响
523家矿山产能利用率 66.1% -0.9个百分点 供应收缩，利多
原煤日均产量 148.5万吨 -1.9万吨 供应收缩，利多
原煤库存 447.4万吨 +6.2万吨 出货放缓，偏空
精煤日均产量 62.5万吨 -2.7万吨 精煤供应下降，利多
精煤库存 193.1万吨 +9.0万吨 下游采购谨慎，偏空

供应端并不是单纯利多，而是“产量下降 + 库存上升”的组合：一方面产量低，支撑焦煤价格；另一方面库存增加，说明高价煤成交和下游采购承接有所减弱。

供应端结论：

产能利用率和产量继续下降，对焦煤下方形成支撑；但原煤、精煤库存同步增加，说明供应紧张并没有完全转化为现货抢购，价格上方仍受成交承接限制。

2. 国内复产与安全监管：山西复产率仍低，结构性紧缺逻辑延续

建信期货7月9日焦炭焦煤日评显示，据Mysteel调研，5月23日起山西炼焦煤矿山共计 171座煤矿 存在停产现象，产能合计 23060万吨；截至7月8日，复产煤矿数量为 98座，产能合计 13495万吨；当日仍处于停产状态的煤矿 73座，产能合计 9565万吨，复产率 58.5%。

这意味着山西供应恢复仍不充分，尤其是优质主焦煤和骨架煤种仍有结构性支撑。建信期货还提到，下半年煤矿安全生产整治或进入常态化阶段，山西、陕西部分煤矿持续检修限产，增添市场对后期煤焦供应的担忧。

对盘面的影响是：

  1. 山西复产率仍不高，供应恢复需要时间；
  2. 停产产能仍接近9565万吨，优质资源偏紧预期延续；
  3. 安监和真实产量监管常态化，限制供应快速释放；
  4. 但库存上升说明下游对高价煤不再主动追买。

3. 蒙煤端：通关下降预期仍支撑盘面，但节日因素偏短期

前期光大期货7月8日矿钢煤焦日报显示，蒙古那达慕大会期间三大口岸将于 7月11日至7月15日闭关5天，7月16日恢复通关；部分蒙古国司机已提前放假回家，本周甘其毛都口岸通关车数预计逐步降至 600车以下。

这一事件继续支撑7月9日焦煤盘面，但需要区分短期和中期：

维度 影响
短期 7月11—15日闭关，通关下降，利多焦煤
中期 7月16日恢复通关后可能集中补发，利多持续性有限
价格端 节前蒙煤价格易稳中偏强
交易端 市场可能提前交易闭关预期，后续关注兑现程度

蒙煤端结论：

那达慕闭关仍是短线支撑，但属于可预期、阶段性因素；7月16日恢复通关后的补发情况，将决定该利多是否转为阶段性利空。

4. 现货端：区域价格分化，主焦煤不是全面上涨

SMM 7月9日焦煤价格页面显示，国内焦煤现货价格出现结构分化：

现货端的核心特点是：

  1. 吕梁、太原、唐山、邯郸等主焦煤价格多数持稳；
  2. 临汾主焦煤、长治主焦煤出现下跌；
  3. 临汾1/3焦煤、瘦煤等部分配煤资源上涨；
  4. 说明现货并非全面走强，而是结构分化。

对盘面的影响是：

主焦煤现货整体仍有支撑，但部分区域主焦价格下调，削弱盘面继续向上突破的信心。

5. 焦炭端：第十轮提涨仍是支撑，但钢厂承接能力偏弱

前期光大期货7月8日报告显示，钢厂对焦炭第十轮提涨暂无回应；第九轮提涨落地后焦企利润修复，目前焦企利润约 50元/吨，焦企开工积极性较好，但多数钢厂仅维持刚需采购焦炭，钢厂盈利率约 40% 左右。

7月9日焦炭期货主力收盘 1945元/吨，下跌 0.08%，与焦煤小幅上涨形成分化。焦炭端说明：

  1. 焦炭现货提涨逻辑尚未完全结束；
  2. 但第十轮提涨承接难度明显上升；
  3. 钢厂利润低位，限制原料上涨空间；
  4. 焦煤上涨如果不能通过焦炭继续传导，下游采购会更谨慎。

焦炭端结论：

焦炭第十轮提涨对焦煤仍有成本支撑，但钢厂未积极回应、焦炭期货未能同步走强，说明焦煤上方受到利润传导受阻的限制。

三、价格驱动逻辑

1. 供应收缩支撑焦煤日盘企稳

523家炼焦煤矿山产能利用率降至66.1%，原煤和精煤日均产量均环比下降，说明国内供应端仍未恢复。这是焦煤7月9日仍能维持1290元/吨附近的主要支撑。

2. 山西停产产能仍高，优质煤结构性支撑存在

截至7月8日，山西仍有73座炼焦煤矿处于停产状态，涉及产能9565万吨，复产率58.5%。这说明优质主焦煤和骨架煤种供应恢复仍慢，对焦煤形成底部支撑。

3. 蒙煤那达慕闭关支撑短线情绪

7月11—15日口岸闭关、甘其毛都通关预计降至600车以下，使短期蒙煤供应扰动仍在，对盘面形成情绪支撑。

4. 库存回升和现货分化压制上方

虽然产量下降，但原煤库存和精煤库存同步增加；SMM现货价格中，临汾主焦煤、长治主焦煤出现下调。这说明下游高价采购并不积极，价格继续上行缺少强成交配合。

5. 焦炭提涨承接受阻限制焦煤突破

焦炭第十轮提涨暂无明显钢厂回应，焦炭期货7月9日也小幅收跌。焦炭端不能强势配合，使焦煤难以有效突破1300元/吨上方。

四、风险点

1. 7月16日后蒙煤集中补发

那达慕闭关支撑短期通关下降，但若7月16日恢复通关后集中补发，前期利多可能转为供应压力。

2. 精煤库存继续回升

523家煤矿精煤库存环比增加9.0万吨，如果后续继续累库，将说明下游采购不足，现货支撑可能减弱。

3. 主焦煤现货继续松动

7月9日临汾主焦煤、长治主焦煤已有下调。如果更多主焦煤区域价格松动，期货高基差支撑将被削弱。

4. 焦炭第十轮提涨落地受阻

钢厂利润低位，第十轮提涨承接难度较大。如果提涨受阻，焦煤成本传导逻辑会下降。

5. 1300元/吨上方压力

7月9日焦煤最高1297.5元/吨，未能有效突破1300元/吨，说明上方仍有明显技术和心理压力。

五、综合判断

7月9日焦煤市场的核心特征是：

供应端仍偏紧，但现货成交和下游承接偏谨慎；盘面维持1290元/吨附近震荡，未能有效突破1300元/吨。

短期判断：震荡偏稳，1300元/吨附近压力仍大

短期看，矿山产量下降、山西复产偏慢、那达慕闭关预期仍支撑焦煤盘面。但7月9日收盘仅上涨0.08%，说明盘面上行动能有限。1290元/吨附近有支撑，但1300元/吨上方需要更多利多催化。

中期判断：供应支撑仍在，但需求端负反馈风险上升

山西复产慢和安监常态化限制供应释放；但库存回升、主焦煤局部下调、焦炭提涨承接受阻、钢厂利润低位，说明需求端负反馈风险正在上升。焦煤中期更可能维持高位震荡，而不是单边上涨。

核心矛盾

当前焦煤核心矛盾是：

矿山产能利用率下降、山西复产率低、蒙煤闭关扰动、优质煤结构性紧缺 对抗 精煤库存回升、主焦煤现货局部松动、下游高价采购谨慎、焦炭第十轮提涨承接受阻。

七、结论

7月9日焦煤主力2609合约收盘 1290元/吨，上涨 1元/吨，涨幅 0.08%，全天最高 1297.5元/吨，最低 1282元/吨，成交量 49.27万手。价格小幅收涨，但未能突破1300元/吨，说明盘面处于高位震荡整理。

利多因素包括：

  1. 523家矿山产能利用率降至66.1%；
  2. 原煤和精煤日均产量继续下降；
  3. 山西仍有73座炼焦煤矿停产，复产率仅58.5%；
  4. 那达慕大会前蒙煤通关预计下降；
  5. 优质主焦煤和骨架煤种仍有结构性支撑。

压力因素包括：

  1. 原煤库存和精煤库存同步回升；
  2. 临汾主焦煤、长治主焦煤现货下调；
  3. 下游焦企高价采购谨慎；
  4. 焦炭第十轮提涨暂无明显钢厂回应；
  5. 焦煤未能有效突破1300元/吨。

综合判断：

7月9日焦煤处于“供应支撑仍在、需求承接偏弱”的高位震荡阶段。短期1290元/吨附近有支撑，但1300元/吨上方压力较大。后续重点看7月11—15日口岸闭关实际影响、7月16日恢复通关后的补发、523家矿山库存变化、主焦煤现货是否继续松动，以及焦炭第十轮提涨能否落地。
