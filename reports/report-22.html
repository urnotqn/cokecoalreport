焦煤期货资讯分析报告

资讯日期：2026年7月3日 品种：大商所焦煤 / 焦煤主力 JM2609 / 焦煤主连 数据来源：东方财富焦煤期货页、东方财富午盘播报、东方财富期货早餐、兰格钢铁网、生意社、新华财经指数等公开信息 整理日期：2026年7月4日

说明：东方财富焦煤期货页为动态行情页，历史行情和资讯需要结合盘后行情表、资讯页和产业资讯交叉核验。7月3日主口径采用兰格钢铁网和生意社盘后行情：焦煤主力JM2609收盘1282元/吨，较上一结算价上涨17元/吨，涨幅1.34%；生意社大连商品交易所行情表显示JM2609前结算1265元/吨，开盘1275元/吨，最高1293元/吨，最低1270元/吨，收盘1282元/吨，结算1280.5元/吨，成交671837手，持仓439114手，持仓减少4987手。

焦煤、焦炭同步反弹，焦煤收复7月1日下跌后的部分失地；但焦煤持仓减少，说明上涨更多偏低位修复和空头回补，持续性仍需观察。

二、主要资讯梳理

1. 供应端：沁源产量自查或再次影响复产节奏

东方财富午盘播报引用Mysteel消息称，长治市沁源县最新下发关于再次在全县煤矿企业开展真实煤炭产量数据自查的紧急通知，要求重新报送 2026年1月1日至2026年6月30日矿井原煤统计表。

据Mysteel调研，截至7月3日，长治沁源县煤矿在产煤矿仍仅一座，产能 150万吨；另外一座产能 210万吨 的煤矿已重新验收结束，正等待验收结果通知。按照最新调研情况，该县其余煤矿暂未收到复产消息。本次产量自查有可能再次影响煤矿复产进度，且下半年即便所有煤矿均恢复生产，也难以达到停产前的产量水平。

这条信息对焦煤偏利多，核心在于：

  1. 山西局部复产进度仍慢；
  2. 产量自查可能延后复产节奏；
  3. 已复产煤矿也未必能恢复至停产前产量；
  4. 优质主焦煤结构性偏紧逻辑继续存在。

供应端结论：

山西供应恢复仍不顺畅，产量自查强化了“复产不及预期”的交易逻辑，对焦煤下方形成支撑。

2. 需求端：铁水仍高，但钢厂盈利率继续下降

东方财富午盘播报中，Mysteel数据显示，247家钢厂高炉开工率 83.99%，环比减少 0.42个百分点；高炉炼铁产能利用率 91.3%，环比增加 0.22个百分点；钢厂盈利率 42.86%，环比减少 8.22个百分点；日均铁水产量 243.25万吨，环比增加 0.30万吨。

指标 变化 对焦煤影响
日均铁水243.25万吨 继续高位 支撑焦煤刚需
高炉开工率83.99% 小幅下降 中性偏空

需求端的关键矛盾是：

铁水高位仍支撑焦煤需求，但钢厂盈利率下降会限制钢厂对焦炭、焦煤继续涨价的接受能力。

3. 钢材端：螺纹表需改善，但库存仍有压力

东方财富期货早餐引用Mysteel数据称，截至7月2日当周，螺纹产量、表需由降转增，厂库由增转降，社库连续两周增加：

螺纹表需改善对黑色系情绪有所支撑，但社库连续两周增加，说明终端需求尚未彻底改善。钢材端对焦煤的影响是：

短期情绪改善，但淡季和库存压力仍在，难以单独支撑焦煤连续上行。

4. 基差端：现货升水仍大，盘面下方支撑明显

生意社7月3日现期表显示，焦煤现货价格为 1911.25元/吨，主力合约价格为 1282元/吨，主力基差为 629元/吨。

这说明现货仍大幅升水期货，期货深贴水格局没有改变。对盘面来说：

高基差限制焦煤继续深跌，但如果现货价格松动，基差支撑也会快速减弱。

5. 焦炭端：第九轮落地后，第十轮提涨计划启动

东方财富午盘播报显示，7月3日山东地区潍坊、滨州、德州、济宁、枣庄、菏泽、日照、泰安、临沂等市场焦炭价格计划提涨，湿熄焦上调 50元/吨，干熄焦上调 55元/吨，自 7月6日0时起执行。

新华财经指数也提到，本期焦炭价格第九轮提涨全面落地，部分焦企有意提出第十轮涨价；但当前钢厂盈利水平较前期有所收缩，后续对焦炭涨价的接受能力下降，预计焦炭价格短期或以稳为主。

焦炭端对焦煤的影响是双向的：

利多方面：

  • 焦炭第十轮提涨计划启动；
  • 焦企成本压力仍需向下游传导；
  • 铁水高位支撑焦炭刚需；
  • 焦煤成本对焦炭仍有支撑。

压力方面：

  • 钢厂盈利率明显下降；
  • 第十轮提涨落地难度高于第九轮；
  • 若钢厂抵触增强，焦企补库焦煤的积极性会下降。

焦炭端结论：

第十轮提涨预期对焦煤偏利多，但钢厂利润下降会限制提涨顺利落地。

三、价格驱动逻辑

1. 沁源产量自查强化供应扰动预期

沁源县重新开展煤矿真实产量数据自查，叠加当前在产煤矿较少，市场担心复产节奏再次放缓。这是7月3日焦煤反弹的重要供给端支撑。

2. 高基差继续托底盘面

焦煤现货1911.25元/吨，JM2609为1282元/吨，基差仍有629元/吨。期货相对现货明显贴水，盘面下方有估值支撑。

3. 铁水继续高位，刚需尚未转弱

247家钢厂日均铁水243.25万吨，仍处高位，说明焦煤、焦炭刚需仍在。只要铁水未明显下降，焦煤难以形成趋势性下跌。

4. 焦炭第十轮提涨计划提振双焦情绪

焦炭第九轮落地后，第十轮提涨计划启动，带动焦炭期货反弹，也对焦煤形成联动支撑。

5. 持仓下降说明反弹持续性仍需验证

虽然焦煤收涨1.34%，但持仓减少4987手，说明上涨可能包含空头减仓和低位修复成分。若后续没有增仓上行配合，反弹力度可能受限。

四、风险点

1. 焦炭第十轮提涨落地不及预期

钢厂盈利率环比下降8.22个百分点至42.86%，对焦炭继续涨价的接受能力下降。如果第十轮提涨落地受阻，会削弱焦煤成本支撑。

2. 现货价格松动

目前焦煤期货重要支撑来自高基差。如果主焦煤或配煤资源现货开始下跌，期货支撑会减弱。

3. 蒙煤通关恢复和口岸成交偏弱

前期甘其毛都扰动缓和后，蒙煤端利多减弱。如果蒙煤通关恢复高位、口岸成交仍偏冷清，会压制期货上方空间。

4. 钢厂利润继续压缩

铁水虽高，但钢厂盈利率下降明显。如果钢厂利润继续恶化，后续可能通过控制焦炭采购、压价原料或检修高炉来反馈到焦煤端。

5. 持仓下降后的反弹持续性不足

7月3日上涨伴随持仓减少，若后续不能转为增仓上行，反弹可能难以持续。

五、综合判断

7月3日焦煤市场的核心特征是：

供应扰动再度强化，盘面低位修复；高基差和高铁水提供支撑，但钢厂利润下降和持仓减少限制上行持续性。

短期判断：偏强震荡

焦煤主力收涨1.34%，价格回到1280元/吨上方。沁源产量自查、焦炭第十轮提涨计划、高基差和高铁水共同支撑盘面，短期情绪较7月1—2日改善。

中期判断：上方空间仍取决于钢厂利润和焦炭提涨

若焦炭第十轮顺利落地，且山西复产继续偏慢，焦煤仍有进一步修复空间。但如果钢厂利润继续下降、现货成交转弱、蒙煤通关恢复高位，焦煤上方空间会受到限制。

核心矛盾

当前焦煤核心矛盾是：

山西复产慢、沁源产量自查、高基差、铁水高位、焦炭第十轮提涨预期 对抗 钢厂盈利率下降、现货高价成交压力、蒙煤扰动缓和、上涨持仓减少。

七、结论

7月3日焦煤主力JM2609收盘1282元/吨，上涨1.34%，是7月1日下跌后的进一步修复。上涨主要来自：

  1. 沁源煤矿产量自查可能再次影响复产；
  2. 在产煤矿仍少，山西局部供应恢复偏慢；
  3. 焦煤现货对期货仍大幅升水，基差约629元/吨；
  4. 日均铁水243.25万吨，刚需仍有支撑；
  5. 焦炭第十轮提涨计划启动，双焦情绪改善。

但需要注意：

  1. 焦煤上涨伴随持仓减少，反弹持续性仍需验证；
  2. 钢厂盈利率降至42.86%，对焦炭继续提涨接受能力下降；
  3. 螺纹社库继续增加，终端需求仍有淡季压力；
  4. 蒙煤扰动缓和后，进口煤端利多减弱；
  5. 如果现货高价成交继续转弱，基差支撑可能下降。

综合判断：

7月3日焦煤处于“供应扰动和高基差支撑下的偏强修复”阶段。短期关注1280元/吨附近能否站稳；后续真正决定方向的变量，是沁源复产进度、焦炭第十轮提涨落地情况、钢厂盈利率和主焦煤现货是否松动。
